# Ticket #002 - Create New User Accounts

## Request
Create Active Directory user accounts for new employees and organize accounts by department.

## Environment
Windows Server Active Directory Domain Controller

Domain:
lab.local

## Actions Taken
Created employee user accounts and placed them into the appropriate Organizational Units:

- IT OU
- HR OU
- Sales OU
- Accounting OU

Configured user accounts with:

- User login names
- Initial passwords
- Password change requirement at next login

## Verification
Confirmed that all user accounts were visible in Active Directory Users and Computers and located in the correct Organizational Units.

## Result
Employee accounts were successfully created and organized.
