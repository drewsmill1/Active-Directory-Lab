# Ticket #006 - Disable User Account

## Request
Disable an Active Directory user account for an employee who no longer requires access.

## Environment
Windows Server Active Directory Domain Controller

Domain:
lab.local

## Actions Taken
Performed account disable procedure:

- Located user account in Active Directory Users and Computers.
- Disabled the user account.
- Verified the account status changed to disabled.

## Verification
Confirmed the user could no longer authenticate using the disabled account.

## Result
User account access was successfully removed.

## Status
Completed
